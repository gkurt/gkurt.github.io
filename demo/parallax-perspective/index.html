Perspective parallax

Scroll down. Every layer below sits at a different translateZ inside one perspective, so the browser scrolls each at a speed set by its depth. Same technique, explained by the thing it describes.

Technique
CSS perspective
parallax.
Set perspective on a parent. Apply translateZ to its descendants. The browser projects them at different depths — and they scroll at different rates.
Mechanism
Three CSS properties.
perspective
Set on the parent. Defines the focal length. Smaller values exaggerate depth.
transform-style: preserve-3d
Tells the parent to keep its children's depth instead of flattening them onto its plane.
translateZ
Moves an element along the depth axis. Negative pushes it back, positive brings it forward.
Each layer must be a descendant of an element with preserve-3d — otherwise the browser flattens its translateZ onto the parent's 2D plane.
Motion
Speed by depth.
speed = perspective ÷ (perspective − translateZ)
